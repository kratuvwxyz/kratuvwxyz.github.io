Another Weather App by Desaign Studio

The app allows the user to see the weather of some selected cities. Users can change locations by pressing the plus icon, enabling the user to type in the city's name. The Another Weather App uses Open Weather API as its source of information for the weather. The HTML Geolocation API gives you your current location weather information.

Tools: HTML5, CSS3, JavaScript, JQuery, Bootstrap, Geolocation API, Open Weather API, Font Awesome Icons, Google Fonts, SVG
